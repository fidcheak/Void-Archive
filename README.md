# Архив Пустоты

Incremental сай-фай хоррор на Godot. Никаких импортированных ассетов — вся графика и UI строятся исключительно кодом и кастомными шейдерами, создавая строгий, минималистичный визуальный стиль.

## Статус проекта: В разработке

**На данный момент полноценного релизного билда нет.** Проект находится на стадии активного прототипирования и разработки. 

Это инди-проект, в который я вкладываю много сил и стараний, стремясь довести до ума каждую механику и передать ту самую гнетущую атмосферу сай-фай хоррора. В процессе разработки я активно использую возможности современных нейросетей для генерации кода, оптимизации алгоритмов и ускорения рутинных задач.

## Особенности

* **Zero-Asset подход:** 100% процедурная визуализация. Все элементы интерфейса, фоны и эффекты отрисовываются математически через код и шейдеры.
* **Движок:** Godot 4.6 (GDScript).
* **Атмосфера:** Медленный idle-прогресс, переплетенный с элементами нагнетающего космического хоррора.

## Запуск

1. Склонируй или скачай этот репозиторий.
2. Открой проект в **Godot 4.x**.
3. Нажми **F5**, чтобы запустить главную сцену из редактора (желательно в режиме совместимости).
