# Changelog

- L0 — фундамент (Godot): автозагрузки, Palette/Theme/CRT, состояние, тик, сейв, форматтер, клик→Данные.
- Фикс пустого серого экрана: в main.gd `set_anchors_preset(PRESET_FULL_RECT)` заменён на `set_anchors_and_offsets_preset(PRESET_FULL_RECT)` для фона, layout-контейнера, ядра, кнопки и CRT-оверлея — без сброса offset'ов контролы оставались 0×0.
- L1 — энергосеть (поток+троттлинг), здания Сканер/Реактор, система покупок, панель зданий, Энергия в топбаре.
- L2 — Вычисления + Суперкомпьютер, система исследований с эффектами-множителями (ветка Машины), обобщённое производство, панель исследований + вкладки.
- L3 — Коррупция (рост от интенсивности производства, бонус к выработке, риск плохих аномалий, стабилизация за Вычисления), Аномалии (signal/glitch, реальные эффекты mult_production/instant_data_seconds, выбор взвешен коррупцией, максимум одна активна), CRT стал пост-процессом с глитчем, гонящимся уровнем коррупции, флаг-крючок void_detected для веток Сознание/Пустота (L5).
